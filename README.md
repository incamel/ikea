# 이케아 Publishing

**이케아 메인 페이지입니다.** 

프로젝트에서 재사용이 많은 CSS 속성(색상, 폰트 크기 등)을 효율적으로 관리하기 위해 <br>
변수화하여 작업했습니다. 이를 통해 수정이 편리하다는 점을 실감했고, <br>
앞으로도 디자인 단계에서부터 퍼블리싱을 염두에 두고 작업할 계획입니다. <br>
구체적으로는 최소한의 폰트 크기, 색상, 여백 값을 활용하여 일관성과 효율성을 높이는 방식을 지향하고 있습니다.
